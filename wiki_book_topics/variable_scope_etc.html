Variable scope etc

Scope

  • Block scope: variable created between curly braces {}, it can't be accessed outside the end }.
  • Function or local scope: Variable created in the function scope can't be accessed outside function.
  • Module: when a large script divide into mulitple files, each file called module. and one module can't access other modules variables.
  • Global: Variables and functions are declared at the top level of the script.

    • Constant: It defines a constant reference to a value

    When to use const:

  • A new Arrary
  • A new Object
  • A new function
  • A new RegExp

    • What is hoisting

    Variables defined with var are hoisted to the top and can be initialized at any time. Meaning: You can use the variable before it is declared:

    carName = "Volvo";

    var carName;

    Constant Objects and Arrays

    You can NOT:
  • Reassign a constant value
  • Reassign a constant array
  • Reassign a constant object
    • You can:
  • Change the elements of constant array
  • Change the properties of constant object
