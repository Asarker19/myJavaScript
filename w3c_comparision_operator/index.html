Comparision Operators only returns true or false

1- Operator == , this only checks the types not the value returns true or false

We are comparing x = 10 and Y ='10', an intiger with a string

2- Operator ===, this checkes value and the types and returns true or false

We are comparing a = 10 and b ='10', an intiger with a string

3- Operator != , this means not equal, it will return true or false

Note: value is same so it returns false

4- Operator !==, this means not equal to value and type, return true or false

Note: value is same but type is different so it returns true

5- Operator < | > | <= | >=| these all return true or false and do not consider data types.

Note: value is same but the type is different, it still compare and returns false
