Ternary also an comparision Operator

Like: if-then-else

  • It takes 3 arguments.
  • Condition or expression ? if Value is ture return this : if value false return this.
    Example: x > y ? x : y;
    Condition is x > y [this is the if condition]
    If this is true, get the value after the ? x [true then get x value ]
    Else take the value after : [else take y value after colon]
    Where x = 11 and y =10.
